1. input() collects data from the user during program execution.

    Syntax: input("What is your name?")

    How it Works:
    User enters data.
    input() captures it.
    The value becomes a string.
    It can be combined with other strings.


2. Input is always stored as a string by default.

3. Input can be stored in variables.

4. Input is dynamic data.

5. User interaction makes programs useful.

6. input() pauses program execution until user enters data.( waits until user click Enter and takes all the input before that click)

7. Input values can later be converted to integers.

    age = int(input("Enter your age?"))
