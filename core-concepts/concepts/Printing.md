1. Printing means displaying output on the screen using the print() function.

2. Text must be inside quotes.

3. Python supports single ' ' and double " " quotes.

4. Triple quotes for multi line (''' or """ both are valid)

single line: print("Hello world!\nHello world!\nHello world!") 

Multi line: 
print("""Hello world!
Hello world!
Hello world!""")

5. Every print() statement prints on a new line by default.

6. Missing quotes or brackets causes syntax errors.

7. print(Hello) --> Error because Hello is treated as a variable.
   print("Hello") --> Hello gets printed in console.
