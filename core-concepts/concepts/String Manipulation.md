
1. String manipulation means changing or formatting text strings.
2. Strings are text data.
3. + operator joins strings.
4. \n creates a line break.
5. Space must be added manually in concatenation.
6. Concatenation works only with strings.


New Line Escape Sequence \n :

    Example: 
    print("Hello world!\nHello world!\nHello world!")
    o/p:  Hello world!
          Hello world!
          Hello world!

String Concatenation :
    
    Example: 

    a. Without Space
    print("Hello" + "Alekhya")
    o/p: HelloAlekhya

    b. With Space
    print("Hello" + " "+"Alekhya")
    o/p: Hello Alekhya

    c. print("Hello" + " Alekhya")
    o/p: o/p: Hello Alekhya
