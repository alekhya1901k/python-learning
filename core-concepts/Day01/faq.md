Q1. What is the purpose of print() in Python?

    It is used to display output to the console.

Q2. Why are quotes required in print()?

    Quotes tell Python that the content is a string.

Q3. Difference between print("5") and print(5)?
    
    "5" → String
    5 → Integer

Q4. What is string concatenation?

    Joining multiple strings into one string.

Q5. What does \n do?

    It inserts a new line.

Q6. Why is spacing important in concatenation?

    Python does not add spaces automatically.

Q7. What datatype does input() return?

    String datatype.

Q8. Can input() take numbers?

    Yes, but they are stored as strings unless converted.

Q9. Why is input() important?

    It allows interaction between user and program.

Q10. Can a variable change datatype?
   
    Yes. this is also called as VAriable Reassignment.
    Example:
    x = 10  
    x = "Hello"

Q11. Difference between = and ==?
	=	for Assignment
    ==	for Comparison

Q12. Difference between Variable and Value
    
    Variable → Container/name that stores data
    Value → Actual data stored inside the variable
    Example:    
    age = 25
    age → Variable
    25 → Value

Q13. Why use variables instead of direct values?

    They make programs reusable and maintainable.

Q14. What are triple quotes used for in Python?

    They are used to create multi-line strings and documentation strings.

Q15. Difference between single quotes and triple quotes?
        
        Single quotes → Single-line strings
        Triple quotes → Multi-line strings
Q16. Are triple single quotes and triple double quotes different?

    No. Both behave similarly for multi-line strings.
