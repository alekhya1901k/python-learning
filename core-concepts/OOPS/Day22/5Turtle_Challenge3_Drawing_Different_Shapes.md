# Turtle Challenge 3 – Drawing Different Shapes

## Quick Short Notes

1. Turtle supports drawing many different shapes using movement commands.
2. Colors improve the appearance and visualization of drawings.
3. Turtle uses color names provided by the Tkinter library.
4. Turtle Graphics is built on top of the Tkinter GUI framework.
5. `forward()` moves the Turtle in its current direction.
6. `right()` rotates the Turtle clockwise by a specified angle.
7. `left()` rotates the Turtle anticlockwise by a specified angle.
8. GUI (Graphical User Interface) allows interaction using windows, graphics, and a mouse, unlike a CLI (Command-Line Interface).
9. The official Turtle documentation lists all available methods and their usage.
10. By combining movement and turning, you can create complex geometric shapes.

## Example

tim.color("blue")
tim.forward(100)
tim.right(90)
