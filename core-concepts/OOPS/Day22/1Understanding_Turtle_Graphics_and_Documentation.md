# Understanding Turtle Graphics & Documentation

## Quick Short Notes

1. Turtle Graphics is a Python module used for creating drawings and graphics.
2. A Turtle object acts like a pen moving on the screen.
3. Every movement can leave a drawing trail.
4. The Screen object controls the drawing window.
5. exitonclick() keeps the window open until clicked.
6. Documentation is the primary source for learning module functions.
7. Methods are accessed using dot notation: object.method().
8. Turtle appearance can be customized using shape() and color().
9. Documentation explains available methods, arguments, and examples.
10. Professional developers rely heavily on official documentation.

## Example

from turtle import Turtle, Screen

tim = Turtle()
tim.shape("turtle")
tim.color("red")

screen = Screen()
screen.exitonclick()

## Quick Summary

1. Turtle is a graphics library.
2. Turtle behaves like a pen.
3. Screen manages the window.
4. exitonclick() prevents immediate closing.
5. shape() changes turtle appearance.
6. color() changes pen/turtle color.
7. Methods belong to objects.
8. Documentation explains module usage.
9. Read documentation before coding.
10. Documentation improves problem-solving skills.
