# Python Tuples & Random RGB Colours

## Quick Short Notes

1. A tuple is an immutable (unchangeable) data type in Python.
2. A list is mutable, meaning its elements can be modified after creation.
3. Tuples are created using parentheses `()`.
4. RGB stands for **Red, Green, and Blue**, the three primary colors used to create digital colors.
5. Each RGB value ranges from **0 to 255**.
6. Turtle Graphics supports RGB colors in addition to named colors.
7. `colormode(255)` enables Turtle to accept RGB values in the range of **0–255**.
8. Tuples are useful for storing constant data that should not be modified.
9. `randint()` is commonly used to generate random RGB values.
10. Immutable data improves reliability by preventing accidental changes.
