# Turtle Challenge 1 – Draw a Square

## Quick Short Notes

1. A square has four equal sides.
2. Every interior angle of a square is **90°**.
3. The Turtle moves forward using the `forward()` method.
4. Turning is done using the `left()` or `right()` methods.
5. Loops eliminate repetitive code.
6. `range(4)` repeats the loop exactly four times.
7. Using loops makes code shorter and easier to read.
8. Geometric shapes are created by combining movements and turns.
9. The Turtle executes instructions one by one in sequence.
10. Breaking a problem into repeated steps simplifies programming.

## Example

for _ in range(4):
    tim.forward(100)
    tim.right(90)

## Quick Summary

1. A square has four equal sides.
2. Use `forward()` to draw each side.
3. Turn **90 degrees** after each side.
4. Repeat the process four times.
5. Loops reduce duplicate code.
6. Fewer lines make code easier to maintain.
7. Turtle draws continuously as it moves.
8. Movement and angles create geometric shapes.
9. `range(4)` matches the four sides of a square.
10. Loops improve efficiency and readability.
