deal_card() → Gives one random card.
calculate_score() → Calculates total score and handles Ace conversion (11 → 1).
compare() → Decides win, lose, draw, or Blackjack.
play_game() → Controls one complete round.
First while loop → User's turn.
Second while loop → Computer's turn.
Final while loop at the bottom → Lets the user play again and again.
return 0 is a special marker used to indicate Blackjack.
_ in for _ in range(2) means the loop variable is not needed.
append() adds new cards to the player's hand.
