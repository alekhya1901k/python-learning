## Day 17 Project: Number Guessing Game

### 10 Quick Summary Points

1. The project is a **number guessing game**.
2. The computer randomly chooses a number.
3. The user tries to guess the number.
4. The game gives hints: **Too high** or **Too low**.
5. The user has limited attempts.
6. Difficulty decides the number of attempts.
7. **Easy mode** gives more attempts.
8. **Hard mode** gives fewer attempts.
9. Functions make the code clean and reusable.
10. The game ends when the user guesses correctly or runs out of attempts.

## Example 1

```text
I'm thinking of a number between 1 and 100.
Choose a difficulty. Type 'easy' or 'hard': easy
You have 10 attempts remaining.
Make a guess: 50
Too high.
Guess again.
```
