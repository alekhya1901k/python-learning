1. What is an if statement in Python?

An if statement is used to check a condition and execute code if the condition is true.

2. What is the purpose of else?

else executes when the if condition becomes false.

3. What type of value does a condition return?

A condition returns either True or False.

4. Why is indentation important in Python?

Indentation defines the block of code under conditions, loops, and functions.

5. Can an if statement exist without else?

Yes, else is optional.

6. What happens if indentation is wrong?

Python throws an IndentationError.

7. How many spaces are recommended for indentation?

4 spaces.

8. Why does Python use indentation?

To define code blocks clearly and improve readability.

9. Can tabs be used instead of spaces?

Yes, but mixing tabs and spaces is discouraged.

10. What is a child block in Python?

Indented code under statements like if, for, or while

11. Difference between = and ==?

= assigns value, == compares values.

12. What does != mean?

Not equal to.

13. What is the output type of comparison operators?

Boolean (True or False).

14. Can comparison operators compare strings?

Yes.

15. Which operator checks equality?

==

16. What is modulo in Python?

Modulo returns the remainder after division.

17. What is the output of 10 % 2?

0

18. How is modulo used to check odd/even?

If number % 2 == 0, number is even; otherwise odd.

19. What is the output of 7 % 4?

3

20. Can modulo return zero?

Yes, when the number divides completely.

21. What is nesting in Python?

Placing one conditional statement inside another.

22. What is elif used for?

To check multiple conditions sequentially.

23. Can multiple elif statements exist?

Yes.

24. Which is better: nesting or elif?

elif is cleaner when checking multiple related conditions.

25. Does every if require else?

No.

26. Difference between multiple if and elif?

Multiple if checks all conditions; elif stops after first true condition.

27. Can multiple if blocks run together?

Yes.

28. Which is faster: elif or multiple if?

elif can be faster because it stops checking after first match.

29. When should multiple if be used?

When conditions are independent.

30. Can else be used with multiple if?

Yes, but usually paired with one specific if.

31. What does and operator do?

Both conditions must be true.

32. What does or operator do?

At least one condition must be true.

33. What does not operator do?

Reverses the Boolean result.

34. What is short-circuit evaluation?

Python stops checking conditions once result is determined.

35. Give a real-world use of logical operators.

Login authentication and age validation systems.
