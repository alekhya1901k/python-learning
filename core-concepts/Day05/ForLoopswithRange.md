3.For Loops with Range

    Quick Short Notes (10 Points)
    1. range() generates a sequence of numbers.
    2. It is commonly used with for loops.
    3. range(start, stop) includes start value.
    4. The stop value is excluded.
    5. range(1, 10) gives numbers 1 to 9.
    6. range() saves memory because it generates numbers lazily.
    7. A third parameter can specify step size.
    8. Range loops are useful for counting operations.
    9. They are heavily used in algorithms and automation.
    10. Range-based loops are faster and cleaner than manual counting.

Basic Example:
# -------------------- range() Example --------------------

# range(1, 10) generates numbers starting from 1 up to 9
# The ending value 10 is NOT included
for number in range(1, 10):

    # Print each number generated by range()
    print(number)

# Output:
# 1
# 2
# 3
# 4
# 5
# 6
# 7
# 8
# 9




# -------------------- Example with Step --------------------

# range(1, 11, 2) means:
# Start from 1
# Stop before 11
# Increase by 2 each time
for number in range(1, 11, 2):

    # Print the current number
    print(number)

# Output:
# 1
# 3
# 5
# 7
# 9




# -------------------- Understanding range() --------------------

# General Formula:
# range(start, stop)

# Mathematical Representation:
# a ≤ range(a, b) < b

# Explanation:
# Start value is included
# End value is excluded

# Example:
# range(2, 6)
# Generates:
# 2, 3, 4, 5




# -------------------- Gauss Challenge --------------------

# Create a variable named total
# It starts from 0 because no addition has happened yet
total = 0

# range(1, 101) generates numbers from 1 to 100
# 101 is excluded
for number in range(1, 101):

    # Add the current number to total
    # Same as: total = total + number
    total += number

# Print the final total after adding all numbers
print(total)

# Output:
# 5050




# -------------------- Mathematical Formula for Gauss Sum --------------------

# Formula:
# n(n + 1) / 2

# Here:
# n = 100

# Calculation:
# 100 × 101 ÷ 2
# = 5050

# This formula gives the sum of numbers from 1 to n
